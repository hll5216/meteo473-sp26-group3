# Meteo 473 Threat Index — Group 3

## Project Description
We developed an index that quantifies good beach day conditions using GFS model data. We pulled the variable quantites of temperature, dewpoint, wind speed, precipitation rate, and total cloud cover. Each variable was weighted within the index at differnet amounts: temperature at 65%, precipitation at 25%, wind speed at 7%, dewpoint at 2% and cloud cover at 1%. The index functions on a 1-10 scale, with numbers closer to 10 representing good conditions for the beach.
